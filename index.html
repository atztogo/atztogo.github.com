    Atsushi Togo¶

    (Computational materials scientist)

    My job is to explore useful solid solid state materials by computer simulation.

    Associated scientific codes (open source projects)¶

    • phonopy: Phonon calculator and analyzer written in Python

    • phono3py: Phonon-phonon interaction and lattice theraml conductivity calculator written in Python

    • spglib: Crsytal symmetry finder written in C

    • kspclib: Collection of functions for crystal reciprocal space in C

    • niggli: Determination of Niggli cell written in C

    Open data project¶

    • phonondb: Phonon calculation database
